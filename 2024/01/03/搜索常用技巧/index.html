搜索常用技巧

  1. 搜索完全匹配的结果
    为词语加商英文””
  2. 从结果中排除
    -，例如 spring -csdn
  3. 组合运算符
    北京|上海 人口 北京OR上海 人口
  4. 通配符
    如 李*杰
  5. 搜索社交媒体
    如 @Martin
  6. 搜索特定文件
    filetype:pdf 红楼梦
  7. 搜索特定网站
    site:baidu.com python spring -site:csdn.com
  8. 查找标题中带有特定词语的网页
    intitle:pyhone -site:csdn
